定义：
块：每次由底层收到的一个数据包
点：每次由AD采样的一个数据点（2个字节）
对：每次由一对AD采样的一对数据点


同步时钟1：
{
	AD层速率：
	约每秒1M=10^6=1000000点
}

同步时钟2：
{
	FPGA层速率：
	约*每秒80块（80帧）
	每块含80*100=8000点=4000对
	约*每秒
	80*80*100=320000点=160000对

	C++层速率
	#更新2016-12-13
	每块采集前100*6=600点=300对
	按100取平均后变为6点=3对
	
	#老版本：
	每块采集前32个点=16对
	约*每秒
	32*80=2560点=1280对
}

同步时钟3：
{
	C#层速率：
	每秒从C++层采集10次，每次3对，取平均后显示
}

显示速率由C#层速率决定，目前相当于每秒采集30对，每3个取平均后按每秒10对的速率显示
